INPUT PARAMETERS

District Name

The User Has to select the name of the District. This Attribute is plays a major role in Prediction

Season

The User has to Enter Season(Rabi/Kharif). It also plays a major attribute in prediction

Crop Type

The User has to enter which crop he want to grow to get the yield for the crop he entered

Irrigation Level

The User should enter Irrigation level i.e the water level given to the crop to get accurate prediction If the User dont have these values the model will take default values

Weather

The User has to enter minimum temperature,maximum Temperature and Rainfall but many users will not have the data of these so we have an inbuilt api such that user just need to enter the District to get these values

Soil Nutrients

Soil Nutrients play a major role in the prediction of the yield.The user has to enter ph value,nitrogen , phosphorous , pottasium , zinc , iron , copper , Managnese If the User doesnt know the values of these parameters the model will take default values



PREDICTION

Prediction With All Parameters

In this interactive prediction model, users enters input parameters. Predictions are based on the entered values. The model generates personalized and insightful predictions using advanced algorithms.

Prediction With Some Parameters

In this interactive prediction model, users are required to input the district, season, and crop name. The temperature data is obtained through an API call, while the soil fertility values are set to default. With these inputs, the model utilizes its advanced algorithms to generate insightful predictions tailored to the specific parameters provided by the user.
